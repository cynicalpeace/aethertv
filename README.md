# AetherTV

**Android TV app for AceStream P2P live TV streaming**

## Installation

1. Download the latest APK from [Releases](https://github.com/cynicalpeace/aethertv/releases)
2. Install via ADB:
   ```bash
   adb connect <device-ip>:5555
   adb install aethertv-v*.apk
   ```
